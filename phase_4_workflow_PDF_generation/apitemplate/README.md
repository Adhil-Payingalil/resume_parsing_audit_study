# APItemplate.io Templates for PDF Generation

This directory contains the HTML, JavaScript, and CSS code used to create resume templates in **APItemplate.io**, which are integrated into the **n8n** workflow for generating PDFs.

The structure is:

- **HTML_JS/**: Contains the HTML structure and embedded JavaScript for each template.
- **CSS/**: Contains the corresponding stylesheets for each template.

## Usage

These templates are used by the `pdf_generation_workflow` in Python (via Jupyter Notebook) to generate PDFs for the audit study. The `template_id` in the filenames corresponds to the template ID in APItemplate.io.
